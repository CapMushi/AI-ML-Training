# Context & Rules for AIML Practical Assignment

## 1. Project Goal
We are implementing a Linear Regression model from scratch (manually) using only NumPy and Pandas to predict housing prices based on features like area and number of bedrooms. 

## 2. Environment & Tech Stack
- IDE: VS Code
- Language: Python 3
- Libraries Allowed: NumPy, Pandas, Matplotlib/Seaborn (for EDA/visualization only). No Scikit-Learn for the model building!

## 3. Core Concepts to Cover (Module Outline Constraints)
The implementation must explicitly demonstrate and cover:
- NumPy arrays, explicit indexing, and slicing.
- NumPy mathematical operations and vectorization/broadcasting (e.g., Matrix multiplication for gradients/normal equations).
- Pandas DataFrames and Series manipulation.
- Data loading, handling missing values, and feature transformation (scaling/normalization).
- Exploratory Data Analysis (EDA).

## 4. Code Style & Difficulty
- Balanced: Not overly simplified (avoid nested 'for' loops where NumPy vectorization should be used), but not overly abstract or hyper-optimized. 
- The goal is deep conceptual understanding of the underlying mathematics and data manipulation.
