# Case Study: Multi-Variable Linear Regression from Scratch

## 1. Problem Statement
Given structural attributes from the Ames Housing Dataset, predict the continuous target variable `SalePrice` without utilizing high-level machine learning libraries (e.g., scikit-learn). The model must dynamically handle features on entirely different geometric scales while optimizing coefficients using matrix algebra.

* **Primary Features Selected:** `GrLivArea` (Continuous above-ground living square footage) and `BedroomAbvGr` (Discrete bedroom count).
* **Target Objective:** `SalePrice` (Raw property value in USD).

---

## 2. Structural Solution Sequence

### Step 1: Exploratory Diagnostics (EDA)
* Quantified feature relationships using a Pearson Correlation Matrix.
* Identified a strong linear boundary profile for `GrLivArea` ($r = 0.71$) and a weak linear profile for `BedroomAbvGr` ($r = 0.17$).

### Step 2: Vectorized Feature Scaling
* Implemented manual Z-score standardization using NumPy matrix broadcasting to center features around a mean of 0 and an exact standard deviation of 1:
  $$x_{\text{scaled}} = \frac{x - \mu}{\sigma}$$
* Preserved target scales to ensure final output interpretability in real-world dollars.

### Step 3: Matrix Bias Augmentation
* Prepended an identity column vector of ones ($x_0 = 1$) to the standardized feature matrix $X$, folding the intercept term $b$ directly into the weights vector $W$. This enables optimized, vectorized loop-free forward passes:
  $$\hat{y} = XW$$

### Step 4: Batch Gradient Descent Engine
* Engineered an iterative optimization routine running over 1,000 epochs with a learning rate ($\alpha$) of 0.01.
* Calculated gradients across the entire 1,460-sample batch simultaneously using matrix transposition and multiplication to minimize the Mean Squared Error (MSE) objective:
  $$\nabla J(W) = \frac{1}{m} X^T (XW - y)$$
  $$W \leftarrow W - \alpha \cdot \nabla J(W)$$

### Step 5: Metric Auditing & Validation
* Measured predictive strength on the training partition using Mean Absolute Error (MAE) and the Coefficient of Determination ($R^2$ Score):
  * **MAE achieved:** \$35,791.59
  * **$R^2$ Score achieved:** 55.77%
* Mapped variance anomalies via residual analysis plots to isolate model limitations on luxury outliers ($>\$500\text{k}$).

### Step 6: Inference Pipeline Deployment
* Constructed a forward matrix-multiplication pipeline to run inference and export predictions on an unlabeled evaluation dataset (`test.csv`).
